## 法律

>《电信和互联网用户个人信息保护规定》：“电信业务经营者、互联网信息服务提供者在用户终止使用电信服务或者互联网信息服务后，应当停止对用户个人信息的收集和使用，并为用户提供注销号码或账号的服务。”
>
>也就是说，用户可以依法要求网站和手机APP注销不再使用的账号，如果对方拒绝，你可以登陆12321.cn，拨打010-12321进行举报。
